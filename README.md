## âš™ï¸ Installation

### 1. Clone the Repository
```bash
git clone https://github.com/AswiniMitikiri/FitFusionAI.git
cd FitFusionAI
```

### 2. Set Up Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3. Set Up Frontend
```bash
cd ../frontend
npm install
npm start
```

### 4. Launch the App
Open your browser and go to:  
`http://localhost:3000`
